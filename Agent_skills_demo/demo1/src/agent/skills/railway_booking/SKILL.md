# 12306铁路查询技能

## 功能范围
- 车站查询：搜索火车站、高铁站信息
- 车票查询：查询火车、高铁班次和余票
- 票务服务：订票、退改签相关操作
- 时刻表查询：列车运行时刻信息

## 使用规范
1. 专门处理铁路交通相关查询
2. 需要明确出发地、目的地、日期等信息
3. 实时票务信息可能会有延迟

## 部分工具说明
- station_search: 车站信息查询
- ticket_query: 车票查询
- booking_management: 票务管理
